## Списки

Чтобы добавить ненумерованныые списки, необходимо пункты выделить звёздочкой (*). Например, вот так:
```sh
* Элемент 1
* Элемент 2
* Элемент 3
```

Чтобы добавить нумерованные списки, необходимо пункты пронумеровать. Например, вот так:
```sh
1. Первый пункт
2. Второй пункт
```
